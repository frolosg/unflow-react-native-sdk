# unflow-react-native-sdk

A React Native SDK for Unflow

## Installation

```sh
npm install unflow-react-native-sdk
```

## Usage

```js
import UnflowReactNativeSdk from "unflow-react-native-sdk";

// ...

const result = await UnflowReactNativeSdk.multiply(3, 7);
```
